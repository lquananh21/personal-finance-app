# Personal Finance Workspace

A workspace for managing personal finance planning, budgeting, tracking, and analysis.

## Structure

| Directory | Purpose |
|-----------|---------|
| `budgets/` | Monthly and annual budget plans |
| `tracking/` | Income and expense tracking |
| `goals/` | Financial goals and milestones |
| `investments/` | Investment portfolio and research |
| `reports/` | Financial summaries and analysis |
