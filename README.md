# Personal Finance Manager — FinanceFlow PRO

A comprehensive single-file personal finance management web application built with vanilla HTML, CSS, and JavaScript.

## Features
- **Dashboard** — Net worth, cash flow, savings rate, expense pie chart, net worth trend line chart
- **Income Tracking** — Multiple sources, frequency options, monthly equivalents
- **Expense Management** — Categorised expenses, search/filter, recurring vs one-time
- **Budget Planning** — Category allocation, budget vs actual, 50/30/20 rule
- **Investment Portfolio** — Asset allocation chart, ROI calculations, diversification view
- **Debt Management** — Payoff timeline, interest tracking, debt burden visualisation
- **Financial Goals** — Progress tracking, savings recommendations
- **Reports & Analytics** — Monthly summary, CSV export, print support

## Usage
Open `index.html` in any modern browser. No server or build step required.

## Tech
- Pure HTML/CSS/JS (no frameworks or libraries)
- Canvas API for charts
- In-memory data with sample data pre-loaded
- Responsive design (mobile-friendly)
- Print-friendly reports
